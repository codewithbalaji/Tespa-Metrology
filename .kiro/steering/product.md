# Product Overview

TESPA is a full-stack e-commerce platform for industrial/technical products with three main applications:

- **Frontend**: Customer-facing website for browsing products, placing orders, and managing accounts
- **Admin**: Dashboard for managing products, orders, enquiries, careers, testimonials, and news
- **Backend**: REST API server handling all business logic, authentication, and data management

## Key Features

- Product catalog with flexible specifications and features
- Shopping cart and order management
- User authentication and authorization
- Admin dashboard with analytics
- Enquiry and career application management
- Testimonial and news management
- AI-powered chatbot integration
- Payment processing (Stripe, Razorpay)
- Email notifications
- Image management via Cloudinary
- SEO optimization with sitemap generation
